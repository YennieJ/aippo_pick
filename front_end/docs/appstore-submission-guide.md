### 개인정보처리방침 URL (필수 — 양쪽 스토어 모두 요구)
앱 내 화면만으로는 안 되고, **외부에서 접근 가능한 URL**이 반드시 있어야 합니다.

선택지:
- GitHub Pages로 정적 페이지 배포 (무료)
- Notion 공개 페이지
- 서비스 웹사이트(`api.aippopick.shop`) 내 `/privacy` 경로

> 현재 앱에는 개인정보처리방침이 인앱 화면(`termAndConditions.tsx`)에만 있습니다.
> URL이 생기면 `app.config.js`에도 추가해두는 게 좋습니다.

### 수출 규정 (암호화)

`ios/app/Info.plist`에 이미 설정됨:
```xml
<key>ITSAppUsesNonExemptEncryption</key>
<false/>
```
→ 별도 ERN(암호화 등록 번호) 불필요. App Store Connect 제출 시 "표준 암호화만 사용" 선택.
